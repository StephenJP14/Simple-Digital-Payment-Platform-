User Login ke login.php
Lalu, login.php di proses ke login_handler.php
Kalo bener, login_handler.php terusin ke home_page.php
Kalo user QRIS, dari home_page.php ke qr_scan.php
Kemudian qr_scan.php ngarahin ke password.php
Lalu password.php bakal check secara asynchronus ke pass_check_handler.php
Dan pass_check_handler.php bakal return true kalo pin bener
Kalau respon pass_check_handler.php bener, maka dari password.php diarahin ke transaction_handler.php


login.php -> login_handler.php -> home_page.php -> qr_scan.php -> (password.php <-> pass_check_handler.php) -> transaction_handler.php
